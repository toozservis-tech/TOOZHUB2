# TooZ Hub 2

Aplikace TooZ Hub 2 je desktop aplikace vyvinutá pro potřeby uživatelů.

## Instalace

1. Stáhněte si projekt z repository.
2. Ujistěte se, že máte nainstalované závislosti (PySide6).
3. Spusťte aplikaci pomocí `python src/app/main.py`.

## Spuštění aplikace

Aplikace můžete spustit pomocí příkazu `python src/app/main.py`.
